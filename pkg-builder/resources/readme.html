You'll need a free Anthropic API key

SmartScreenshots uses Claude AI to read each screenshot and generate a descriptive filename. To do this, it needs an API key — a private credential that identifies your account when the script calls the Claude API.

At the end of this installer, a dialog will appear asking you to paste your key. Get it ready now:

1
Go to console.anthropic.com/keys in your browser and sign in (or create a free account).
2
Click Create Key, give it a name like "Screenshot Renamer", and copy the key — it starts with sk-ant-.
3
Keep it on your clipboard. When the installer finishes, a dialog will appear — paste the key there to complete setup.
Cost: Claude's API charges per image analysed. At current rates, renaming 100 screenshots costs roughly $0.01–$0.05 depending on image size. You can set a usage limit at console.anthropic.com/settings/limits.

Privacy: Screenshot images are sent to Anthropic's API for analysis and are not stored or used for training by default. See anthropic.com/privacy.
