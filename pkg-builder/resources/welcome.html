SmartScreenshots

This tool automatically gives your screenshots meaningful names — so instead of Screenshot 2026-02-17 at 1.05.48 PM.png you get something like 2026-02-17_zoom-weekly-team-standup.png.

Here's how it works:

  • macOS saves new screenshots to your ~/Desktop/Screenshots folder
  • Once an hour, this tool scans that folder for any new unprocessed screenshots
  • Each one is sent to Claude AI, which reads the image and generates a short descriptive title
  • The file is renamed to YYYY-MM-DD_descriptive-title.png automatically

The renamer runs silently in the background as a macOS LaunchAgent — no app to keep open, no menu bar icon.

What gets installed: a Python script in /usr/local/smartscreenshots/, a virtual environment with the Anthropic SDK, and a LaunchAgent in your ~/Library/LaunchAgents/ folder.
